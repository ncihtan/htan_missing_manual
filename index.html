# WELCOME

The HTAN Manual provides an overview of Human Tumor Atlas Network (HTAN) data, including details regarding data access and data contribution. If you have any questions regarding the manual or HTAN data, please contact us: HTAN Help Desk.

The manual can be found at https://docs.humantumoratlas.org/.

If you have feedback for this manual, including broken links or incorrect information, please submit a ticket to the HTAN Help Desk.

# Quick Start

If you are interested in data reuse, the following pages may be helpful:

  • What is HTAN?
  • How is HTAN Data structured?
  • Do Phase 1 and Phase 2 HTAN data differ?
  • How can I explore HTAN data?
  • How do I obtain a copy of the HTAN data?

Questions or problems accessing the data? Please contact us: HTAN Help Desk.

Only HTAN Members are able to contribute data to HTAN.

If you are a data contributor, the following pages may be most useful:
