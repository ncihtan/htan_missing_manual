# Submitting Publications

The HTAN Data Coordinating Center (DCC) supports HTAN Center publications in the following ways:
* All access-controlled data is submitted under the same dbGAP Accession Study Number: <a href="https://www.ncbi.nlm.nih.gov/projects/gap/cgi-bin/study.cgi?study_id=phs002371" target="_blank" rel="noopener noreferrer">phs002371</a>.
* The HTAN Portal serves as the main data access site. It can be referenced by its Research Resource Identifier (RRID): https://scicrunch.org/resolver/SCR_023364/.
* [Recommended data availability language](#data-availability-statements) is provided on this page.
* After a manuscript is published, an HTAN Center can work with the DCC to [create a Publication Page](#creating-an-htan-portal-publication-page) on the HTAN Portal. A Publication Page provides links to all data used in the publication.

!!! In order to support data sharing and public data access, the DCC encourages authors using HTAN data to either:
* use HTAN identifiers in their publication; or 
* provide a lookup table in the publication to map publication identifiers to HTAN identifiers.
!!!

!!!tip
If you are planning a publication, please notify your [Data Liaison](../data_submission/Data_Liaisons.md) as early as possible to ensure data release planning can best accommodate publication timelines.
!!!

Please note: **The DCC does not issue Digital Object Identifiers (DOIs)** for a specific publication or study level datasets.

## Creating an HTAN Portal Publication Page
To facilitate data sharing and adherence to FAIR (Findability, Accessibility, Interoperability, and Reusability) principles, the HTAN portal provides links to specimen files used in publications. Currently, the HTAN Data Coordinating Center (DCC) faciliates this linking once provided the appropriate information by HTAN Centers.  To submit publication information, HTAN Center's should contact Alex Lash at alexl@ds.dfci.harvard.edu. 

To see examples of Publication Pages from HTAN Phase 1, please see the Publication Tab on the <a href="https://humantumoratlas.org/explore" target="_blank" rel="noopener noreferrer">HTAN Portal Explore page</a>
